Monster Size

The monster size is determined by the basic size and variable limit width set for each quest.
Basic size % × fluctuation value % (within fluctuation limit range)

The basic size is set for each quest, and is expressed in % inside.
This is multiplied by the randomly determined variation value % (varying up and down around 100), and the resulting % multiplied by the size of the monster at 100% determines the size.

The variation table is first selected from one of the following seven types.
